Overview

This class acts as a controller to perform location fetches against a Realm object that contains latitude and longitude values (the object must also contain a primary key).

The controller performs the fetch and creates ABFAnnotations. If a sort descriptor is specified the resulting objects will be sorted by distance.

Finally, the controller can also cluster the objects based off the visible map rect size and zoom level. If a clustering fetch is performed, the resulting ABFAnnotations will be type ABFAnnotationTypeCluster and contain all of the objects in the cluster.
